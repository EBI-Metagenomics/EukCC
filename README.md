## Contributing
To contribute to this project setup the build enviorement like this:
   
```
git clone https://github.com/openpaul/eukcc
cd eukcc
pip install -r requirements-dev.py
pre-commit install
```

## FAQ

**GeneMark-ES throws a lot of errors**
